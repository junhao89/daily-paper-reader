## 摘要
世界动作模型（WAMs）通过联合建模视觉状态转换和机器人动作，为机器人操作提供了一种有前景的范式。然而，现有的WAMs受限于有限的时间上下文、粗粒度的片段级语言监督以及主要基于文本的条件设置，这阻碍了任务进度追踪和细粒度的语言-视频-动作对齐，同时限制了视觉上下文推理和跨实体迁移。本文中，我们提出了WorldScape Policy 2.0，一种具有推理增强长短期记忆的可控WAM。其因果短期视觉记忆提供最近观测作为DiT预填充，以保留局部交互动态；而其长短期事件记忆将历史VLM输出组织为全局历史、局部活跃和事件边界表示，用于进度感知检索。检索到的历史增强感知和自回归生成的规划标记，为自主规划提供了隐式子目标条件；语义强制进一步将事件级指令语义注入这一潜在规划路径。为了建立细粒度多模态可控性，我们构建了Event-5M，一个事件驱动的具身预训练数据集，包含近500万个事件片段，并对齐了动作轨迹、片段级任务指令、片段级子任务描述、目标图像和视频演示。这些设计为高级指令下的自主规划和来自细粒度文本、目标图像或视频上下文提示的可控执行提供了统一接口。在仿真和真实世界平台上的实验展示了其在长时域自主规划、细粒度指令跟随和上下文适应方面的卓越能力。

## Abstract
World Action Models (WAMs) offer a promising paradigm for robotic manipulation by jointly modeling visual state transitions and robot actions. However, existing WAMs are constrained by limited temporal context, coarse episode-level language supervision, and predominantly text-only conditioning, which hinder task-progress tracking and fine-grained language-video-action grounding while limiting visual-context reasoning and cross-embodiment transfer. In this paper, we introduce WorldScape Policy 2.0, a controllable WAM with reasoning-augmented long short-term memory. Its causal short-term visual memory supplies recent observations as DiT prefill to preserve local interaction dynamics, while its long short-term event memory organizes historical VLM outputs into global-history, local-active, and event-boundary representations for progress-aware retrieval. The retrieved history augments perception and autoregressively generated planning tokens, yielding an implicit subgoal condition for autonomous planning; semantic forcing further transfers event-level instruction semantics into this latent planning pathway. To establish fine-grained multimodal controllability, we construct ManipEvent-5M, an event-grounded embodied pretraining dataset containing nearly 5 million event segments with aligned action trajectories, episode-level task instructions, segment-level subtask captions, goal images, and video demonstrations. These designs provide a unified interface for autonomous planning from high-level instructions and controllable execution from fine-grained text, goal-image, or video-context prompts. Experiments in both simulation and real-world platforms demonstrate superior capabilities in long-horizon autonomous planning, fine-grained instruction following and in-context adaptation.

  - **受限的交互接口**：现有WAMs大多仅支持文本条件，无法利用目标图像、演示视频等更自然的视觉提示，限制了视觉推理和跨本体迁移。
- **整体含义**：本文旨在将WAM从短时预测模型升级为可控制长时规划的策略，需要解决“保留什么历史信息”、“如何通过细粒度监督对齐原子动作”、“如何表达超越语言的任务意图”三个互相关联的问题。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 核心思想
- 提出**WorldScape Policy 2.0**，一种具有**推理增强长短期记忆**的可控WAM。
  - **短期视觉记忆**：因果DiT保留最近观测作为视觉预填充，维护局部交互动态。
  - **长期事件记忆**：VLM分支将历史时间块组织为**全局历史、局部活跃、事件边界**三种记忆视图，并进行进度感知检索。
  - **隐式子目标规划**：检索到的记忆增强当前感知和自回归生成的规划标记（CoT），形成隐式条件，通过**语义强制（Semantic Forcing）**将事件级标签转移到自主规划路径。
- 构建**ManipEvent-5M**数据集：近500万个事件片段，对齐动作轨迹、层级文本指令、目标图像、视频演示，支持多模态可控性。

### 2.2 关键技术细节

#### （1）统一动作表示与本体适配器
- 动作表示：每臂使用相对于块第一帧的delta位置（3D）、连续6D相对旋转、绝对1DoF夹爪命令；双臂为20维向量。
- 每个本体有自己的动作编码器 \(E_a^{(e)}\) 和动作解码器 \(D_a^{(e)}\)，共享DiT主干。

#### （2）多模态提示编码
- 细粒度子任务指令通过T5编码为 \(s_t\)，直接注入DiT交叉注意力。
- 高级任务指令由VLM结合当前观测和事件记忆，生成记忆增强的VLM标记 \(\hat{q}_t\)（用于自主规划）。
- 目标图像和视频演示通过VAE编码为持久视觉预填充。

#### （3）推理增强长短期记忆
- **VLM推理**：一次性前向传播得到感知隐藏状态 \(u_t\) 和自回归规划标记 \(r_t^{1:K}\)，拼接为当前推理隐变量 \(q_t\)。
- **事件记忆队列**：维护历史 \(q_j\)，通过注意力池化压缩感知标记为“gist tokens”，保留规划标记。构建三个视图：
  - 全局历史隐变量 \(m_t^{\text{gh}}\)
  - 局部活跃隐变量 \(m_t^{\text{la}}\)（最近 \(S_e\) 个块）
  - 事件边界隐变量 \(m_t^{\text{eb}}\)（通过余弦相似度变化检测高变化时刻，选择最多 \(S_b\) 个边界）
- **检索与门控融合**：将三个视图与压缩历史库拼接，通过交叉注意力检索，再经学习门控 \(\gamma_t = \sigma(W_g[q_t; \tilde{m}_t] + b_g)\) 控制注入量：\(\hat{q}_t = q_t + \alpha \gamma_t \odot \tilde{m}_t\)。

#### （4）隐式子目标潜在规划与语义强制
- 语义强制损失：将 VLM 推理摘要 \(\bar{q}_t\) 通过可训练投影器 \(\phi_q\) 映射到T5语义空间，与固定T5摘要 \(\bar{s}_t\) 计算余弦相似度损失：
  \[
  \mathcal{L}_{\text{sem}} = 1 - \tilde{s}_t^\top \tilde{q}_t
  \]
  其中 \(\tilde{s}_t = \text{sg}(\bar{s}_t / \|\bar{s}_t\|_2)\)，\(\tilde{q}_t = \phi_q(\bar{q}_t) / \|\phi_q(\bar{q}_t)\|_2\)。
- 模式选择：细粒度指令跟随使用T5嵌入 \(s_t\)；自主规划使用记忆增强VLM标记 \(\hat{q}_t\)。

#### （5）联合视频-动作流匹配
- 训练目标：
  \[
  \mathcal{L} = \mathcal{L}_{\text{act}} + \lambda_w \mathcal{L}_{\text{world}} + \lambda_s \mathcal{L}_{\text{sem}}
  \]
  其中 \(\mathcal{L}_{\text{act}}\) 和 \(\mathcal{L}_{\text{world}}\) 分别为动作和未来视频隐变量的流匹配损失（原始动作空间）。

#### （6）三阶段课程训练
- **阶段1**：事件驱动多模态WAM预训练（仅使用 \(\mathcal{L}_{\text{act}} + \lambda_w \mathcal{L}_{\text{world}}\)），建立细粒度语言-视频-动作对齐和短期视觉记忆。
- **阶段2**：记忆感知中训练（引入VLM规划分支、事件记忆和语义强制 \(\mathcal{L}_{\text{sem}}\)），实现自主规划。
- **阶段3**：下游交互后训练，适配特定任务和本体。
