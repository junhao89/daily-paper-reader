## 摘要
类似于人类顺序学习新任务的自然能力，配备视觉-语言-动作（VLA）模型的机器人在部署到开放世界环境时应具备终身学习能力，以学习新任务。然而，最近提出的大多数终身学习模型旨在有效学习当前任务（可塑性）或保持先前任务的高准确性（稳定性），而可塑性与稳定性的权衡在机器人操作模型中仍未得到很好解决。为应对这一基本挑战，我们提出了一种适用于机器人操作的高效缓存终身视觉-语言-动作学习框架（即LifelongVLA），该框架通过双时间尺度适应机制缓解可塑性与稳定性的权衡，同时通过高效缓存重放策略实现低成本机器人部署。具体而言，我们提出了一种双时间尺度LoRA门控模块，将VLA适应分解为两个轻量级路径：一个用于可塑性的短期适配器和一个用于稳定巩固的长期适配器。这些路径通过任务感知门集成，从而实现对可塑性与稳定性权衡的显式控制。在技能重放阶段，提出了一种高效缓存随机重放策略，无需存储完整轨迹即可保留更均衡的保留信号。最后，实验表明LifelongVLA优于现有基线，展示了高效的技能扩展、对已学操作行为的稳健保留，以及在实际部署到xArm机器人时减少对再训练的依赖。

## Abstract
Similar to the natural capabilities of humans to sequentially learn new tasks, robots with Vision-Language-Action (VLA) models should possess lifelong learning ability to learn a new task when deployed in open-world environments. However, most recently proposed lifelong learning models aim to effectively learn the current task (plasticity) or maintain high accuracy on previous tasks (stability), while the plasticity-stability trade-off remains largely unsolved in robotic manipulation models. To address this fundamental challenge, we propose a cache-efficient lifelong Vision-Language-Action learning framework for robotic manipulation (i.e., LifelongVLA), which alleviates the plasticity-stability trade-off with a dual-timescale adaptation mechanism while achieving low-cost robotic deployment with a cache-efficient replay strategy. More concretely, we propose a dual-timescale LoRA gating module to decompose VLA adaptation into two lightweight pathways: a short-term adapter for plasticity and a long-term adapter for stable consolidation. These pathways are integrated via a task-aware gate, enabling explicit control of the plasticity-stability trade-off. In the skill replay phase, a cache-efficient stochastic replay strategy is proposed to preserve more balanced retention signals without full-trajectory storage. Finally, experiments show that LifelongVLA outperforms existing baselines, demonstrating efficient skill expansion, robust retention of learned manipulation behaviors, and reduced reliance on retraining for real-world deployment on an xArm robot.

- **研究动机**：受人类顺序学习能力的启发，需要设计一种能够持续获取新操作技能同时不遗忘已学技能的终身VLA学习框架。
- **整体含义**：本文提出 **LifelongVLA**，旨在通过双时间尺度适配机制和缓存高效重放策略，实现低成本的机器人终身部署，解决塑性-稳定性权衡问题。

## 2. 方法论：核心思想、技术细节、算法流程

### 2.1 核心思想
- 将VLA适配分解为两个轻量级路径：**短期适配器**负责快速学习新技能（塑性），**长期适配器**负责稳定巩固旧技能（稳定性），并通过**任务感知门**动态组合，显式控制塑性-稳定性权衡。
- 引入**缓存高效随机重放**模块，仅存储少量技能的冻结前缀令牌（stop-gradient prefix tokens）和紧凑状态-动作监督，并在重放时重新采样扩散变量重构后缀，避免存储完整轨迹，降低存储和计算开销。

### 2.2 关键技术细节

#### 双时间尺度LoRA门控（Dual-Timescale LoRA Gating）
- 对于每个适配层，定义短期LoRA更新 `ΔW_sh` 和长期LoRA更新 `ΔW_lg`。
- 短期LoRA仅由当前任务损失 `L_new` 更新，获得高塑性；长期LoRA仅由重放损失 `L_rep` 和蒸馏损失 `L_dist` 更新，并设置较小学习率 `η_lg < η_sh`，使长期路径演化更慢，保持稳定。
- 门控上下文 `c` 由冻结的前缀特征池化得到（避免循环依赖），通过轻量网络预测长期路径的贡献权重 `α`。
- 最终权重为：`ΔW_composed = (1-α)*ΔW_sh + α*ΔW_lg`，实现单次前向传播的权重级组合。

#### 缓存高效随机重放（Cache-Efficient Stochastic Replay）
- 每个新任务随机选择少量样本（配额 `ct=50`），仅存储其**停止梯度的前缀令牌**和状态-动作对，不存原始图像或完整轨迹。
- 重放时，从缓存中取出样本，**重新采样扩散变量**（时间步 `τ'` 和噪声 `ϵ'`），并使用当前模型重新计算后缀令牌，产生多样的随机去噪实例。
- 采用掩码损失：当前样本和重放样本混合，分别计算归一化的 `L_new` 和 `L_rep`，并加入蒸馏损失 `L_dist`（使用上一任务快照作为教师），总损失为 `L = L_new + λ L_rep + β L_dist`。

### 2.3 算法流程
1. 冻结预训练VLA骨干（如PaliGemma + Gemma 2B + 300M decoder）。
2. 对新任务t的数据，通过短期LoRA更新当前任务损失。
3. 从缓存中采样旧任务样本，重新计算后缀，通过长期LoRA更新重放和蒸馏损失。
4. 门控根据样本特征动态组合短期和长期LoRA权重，单次前向传播预测动作。
5. 将新任务的少量样本前缀和状态-动作对存入缓存（使用水库抽样）。
6. 重复步骤2-5直到所有任务训练完毕。
