## 摘要
视觉-语言-动作（VLA）模型在视觉运动策略学习中取得了令人瞩目的成果，但其本质上仍是被动的——即仅根据当前观测和语言映射到动作，而缺乏对世界动态的显式前向预测。现有的视觉预见方法能够预测未来视觉状态，但缺乏明确的运动引导：它们显示目标位置，却未给出如何到达该处的方法。我们认为，未来特征预测与稀疏点跟踪具有天然的互补性：前者提供目标状态，后者则捕捉到达目标状态的连续运动路径。为此，我们提出FoMoVLA框架，通过联合学习未来特征预见与稀疏二维点跟踪，用显式的时空监督增强VLA表示，从而改进连续动作策略。FoMoVLA引入了紧凑的预见标记以解码未来特征状态，解码稀疏的时序二维点轨迹以建模紧凑的几何运动，并通过轻量化的未来条件交叉注意力模块将两者耦合，实现预测状态与点动力学之间的一致性推理。在LIBERO、RoboCasa GR-1桌面及LIBERO-Plus上的大量实验表明，该方法达到了最先进的性能，并具有强大的零样本泛化能力。项目页面见https://liauto-research.github.io/FoMoVLA。

## Abstract
Vision-Language-Action (VLA) models have achieved impressive results in visuomotor policy learning, yet remain fundamentally reactive, mapping current observations and language to actions without explicit forward prediction of world dynamics. Existing visual foresight methods predict future visual states but lack explicit motion guidance: they show where to go but not how to get there. We argue that future feature prediction and sparse point tracking are naturally complementary: the former provides the goal state, while the latter captures the continuous motion path toward it. We propose FoMoVLA, a framework that augments VLA representations with explicit spatio-temporal supervision by jointly learning future feature foresight and sparse 2D point tracking, enhancing the continuous action policy. FoMoVLA introduces compact foresight tokens to decode future feature states, decodes sparse temporal 2D point trajectories to model compact geometric motion, and couples both through a lightweight future-conditioned cross-attention module that enables consistent reasoning between anticipated states and point dynamics. Extensive experiments on LIBERO, RoboCasa GR-1 Tabletop, and LIBERO-Plus demonstrate state-of-the-art performance and strong zero-shot generalization. Project page is available at https://liauto-research.github.io/FoMoVLA.

## 1. 核心问题与整体含义（研究动机和背景）

现有视觉-语言-动作（VLA）模型虽然性能强大，但本质上是“反应式”的——它们仅根据当前视觉观测和语言指令直接映射到动作，缺乏对世界动态变化的显式前向预测。近年来的视觉预见方法试图预测未来视觉状态（如像素级或特征级），但这类方法只告诉模型“最终要到达什么状态”，却没有给出“如何通过连续运动到达该状态”的具体引导。作者指出，未来特征预测（提供目标状态）与稀疏点跟踪（提供连续运动路径）具有天然的互补性：前者给出全局目标，后者捕捉局部几何运动。因此，论文提出将两者联合学习，并通过显式的条件交互（未来条件交叉注意力）实现协同，从而增强VLA模型的时空推理能力和动作生成质量。

## 2. 方法论

### 核心思想
在VLA模型（基于StarVLA-GR00T）的训练过程中，注入三个互补的辅助分支：
- **点跟踪分支**：从VLM图像标记的隐状态预测稀疏2D点轨迹（“如何运动”），使用冻结的CoTracker-v3作为教师生成监督信号。
- **未来特征预测分支**：引入K个可学习的`<Foresight>`标记，其隐状态通过EMA教师网络预测动作块最后一帧的视觉特征（“最终去哪里”），并通过轻量MAE解码器进行重构。
- **未来条件交叉注意力模块（FCCA）**：将预测的未来特征作为条件，注入到点跟踪分支的输入中，使运动预测基于预期的未来状态。

### 关键技术细节
- **输入序列重排**：将文本指令放在图像标记之前，确保图像标记的隐状态能感知完整语言指令（因果注意力中后续标记不可见前序标记）。
- **稀疏点选择**：图像Patches的每个网格中心映射到原始像素坐标，得到N=64个查询点，每个点对应一个图像标记，监督其在动作块时长内的位移和可见性。
- **损失函数**：总损失 = 动作流匹配损失 + λ1×未来特征余弦相似度损失 + λ2×（位移回归损失 + 可见性分类损失 + λsmooth×轨迹平滑损失）。平滑项为二阶时间差正则项，防止轨迹突变。
- **EMA教师**：视觉编码器的动量平均副本用于生成稳定的未来特征目标，不参与梯度更新。
- **FCCA模块**：多头注意力，输出投影初始化为零，保证训练开始时不影响预训练表示，随着训练逐渐注入未来信息。
- **推理时**：所有辅助分支均丢弃，仅保留额外的`<Foresight>`标记作为输入（增加9.4ms延迟和0.1GB显存），无额外参数量。

### 算法流程简述
1. 当前帧图像和语言指令输入VLM，序列为 `[指令, 图像标记, <Foresight>标记, 动作标记]`。
2. 图像标记的隐状态通过点跟踪头部预测N个点的T帧位移和可见性，与CoTracker生成的GT计算损失。
3. `<Foresight>`标记的隐状态经过MAE解码器重构未来帧的视觉特征（M个Patches），与EMA教师提取的GT特征计算余弦相似度损失。
4. FCCA模块将`<Foresight>`隐状态作为Key/Value，图像标记隐状态作为Query，生成增强的运动特征用于点跟踪。
5. 动作头采用条件流匹配（DiT）预测动作块，损失为速度场回归。

## 3. 实验设计

### 数据集与Benchmark
- **LIBERO**：四个子集（Spatial, Object, Goal, Long），每个含10个任务，每个任务50个演示。评估成功率（20次rollout/任务，动作块长度T=8）。
- **RoboCasa GR-1 Tabletop**：24个桌面拾放任务，每个任务1000个演示（共24,000），评估成功率（50次rollout/任务，T=16）。使用类人机器人GR-1的自我中心视角。
- **LIBERO-Plus**：LIBERO的分布外扩展，包含7种扰动维度（视角、机器人状态、语言、光照、背景、噪声、布局），共10,030个实例。零样本评估，不进行微调。

### 对比方法
- **通用VLA/世界动作模型**：GR00T-N1.5、π0、X-VLA、LangForce、Spatial Forcing、Fast-WAM、LingBot-VA、Cosmos Policy等。
- **未来预测方法**：WorldVLA、DreamVLA、UniVLA、LaRA-VLA、HiF-VLA。
- **点跟踪方法**：FlowVLA、GeoPredict、JOPAT。
- **基线与消融**：StarVLA-GR00T作为基础骨干，逐步添加未来预测、点跟踪、FCCA。

### 实验结果摘要
- LIBERO：完整方法平均成功率98.8%，优于所有对比方法（最高第二为98.5%）。在Long子集上提升显著（+1.8% vs 无FCCA）。

## 6. 主要结论与发现

- 未来特征预测和稀疏点跟踪这两种辅助目标各自独立提升VLA性能，但它们的组合必须通过显式条件交互（FCCA）才能实现协同，简单多任务相加效果有限。
- FCCA模块使得点跟踪预测更准确（ATE-moving降低39.5%，Survival@10px提升17.3个百分点），验证了预测的未来状态为运动预测提供了强先验。
- 稀疏的8×8网格（64点）比更密的16×16网格效果更好，说明稀疏监督足以捕捉任务相关运动，过密反而引入冗余噪声。
- 紧凑的K=16未来特征标记（通过MAE解码器）优于直接使用所有图像标记（Direct-64），鼓励模型提取全局场景变化而非局部纹理。
- FoMoVLA可以迁移到不同的动作头（流匹配、朴素扩散等），具有框架无关性。
- 零样本泛化能力显著，尤其在语言和背景扰动上，但在视角和机器人初始状态变化上改进有限（受限于2D运动监督）。

## 8. 不足与局限

- **2D局限性**：当前点跟踪仅在2D图像空间中进行，无法建模3D几何变化和深度位移。对于视角变化较大的场景（如LIBERO-Plus中的相机扰动），泛化能力较弱（仅提升10个百分点左右），论文明确指出这是局限性。
- **依赖第三方跟踪器**：CoTracker-v3作为教师，其准确性直接影响监督信号质量；虽然冻结且不参与梯度，但跟踪器本身可能在分布外场景下存在误差。
- **缺乏3D/深度信息**：无法处理需要深度推理的任务（如堆叠、精确放置），未来方向是引入3D运动预测。
- **实验规模**：虽然覆盖了LIBERO和RoboCasa，但机器人平台种类仍有限（固定基座机械臂和类人Robot GR-1），未在真实世界大规模多样性数据集（如Bridge Data v2）上验证。
- **未来特征预测的语义粒度**：仅预测特征空间（余弦相似度），而非像素级，可能丢失精细纹理信息，但对于操作任务似乎足够。
- **消融中未提供统计显著性**：成功率可能具有随机性，尽管多次rollout（20/50次），但未给出置信区间或标准误差。
