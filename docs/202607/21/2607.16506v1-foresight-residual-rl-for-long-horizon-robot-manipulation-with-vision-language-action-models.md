## 摘要
视觉-语言-动作（VLA）策略提供了强大的通用操作先验，但在高精度、接触密集的装配任务中常常失败，原因是长时域信用分配和子任务耦合：当前技能在几何上成功的状态可能对下游技能是脆弱的。我们在冻结的VLA基础策略上展示了残差强化学习（RL）中的这种失败模式：恒定的稀疏成功奖励单独改进了每个子任务，但当技能链式执行时几乎没有收益，因为终端状态质量不受控制。我们提出远见残差强化学习，通过用离线估计的远见值——当前子任务终端状态下未来子任务成功的概率——增强每个子任务的稀疏成功奖励，来优化交接质量。具体地，我们（i）从基础策略终端状态的图像中训练一个视觉远见预测器，使用下游回滚统计数据标注，以及（ii）通过反向远见归纳法训练残差策略，将预测器输出作为奖励乘数。在Isaac Gym中的三相扳手螺母拧紧装配任务（抓取、移动插入、旋转）上，我们的方法实现了85.6%的全任务成功率，优于标准子任务残差RL（54.5%）和VLA基线，同时保持每个子任务的成功率不变。这些结果强调，提高长时域性能需要塑造每个子任务产生哪些成功状态，而不仅仅是是否成功。

## Abstract
Vision-Language-Action (VLA) policies offer strong general-purpose manipulation priors, but often fail on tight-tolerance, contact-rich assembly due to long-horizon credit assignment and subtask coupling: a state that is geometrically successful for the current skill can be brittle for downstream skills. We show this failure mode in residual reinforcement learning (RL) over a frozen VLA base policy: constant sparse success rewards improve each subtask in isolation yet yield little or no gain when skills are chained, because terminal state quality is uncontrolled. We propose Foresight Residual RL, which optimizes handoff quality by augmenting each subtask's sparse success reward with an offline-estimated foresight value -- the probability of future subtask success conditioned on the terminal state of the current subtask. Concretely, we (i) train a visual foresight predictor from images of terminal states of the base policy, labeled using downstream rollout statistics, and (ii) train residual policies via backward foresight induction, using the predictor output as a reward multiplier. On a three-phase wrench-based nut-tightening assembly task in Isaac Gym (grasp, move-insert, rotate), our method achieves 85.6% full-task success, outperforming standard subtask residual RL (54.5%) and VLA baselines, while leaving per-subtask success unchanged. These results highlight that improving long-horizon performance requires shaping which successful states are produced at each sub-task, not only whether success occurs.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：视觉-语言-动作（VLA）模型在通用操作中表现良好，但在高精度、接触密集的长时间序任务（如工业装配）中常因**子任务耦合**而失败。具体表现为：当前子任务成功后产生的终端状态虽然满足本子任务的几何成功条件，但可能对下游子任务非常脆弱（如不稳固的抓取或过浅的插入），导致链式执行时整体任务失败。
- **背景问题**：现有两种主要策略都存在缺陷：①独立训练各子任务策略（包括标准VLA微调和残差RL），完全不考虑下游影响；②分布匹配方法（如对抗性技能链），仅优化状态分布的“重叠”而非实际任务成功。论文指出，真正需要的是**远见价值函数**（foresight value function），即当前终端状态导致下游成功的概率，而这是被前两类方法忽略的关键信息。
- **整体含义**：要提高长时域组合性能，**不能只追求每个子任务是否成功，更要优化“成功状态的质量”**——即产生那些更有利于后续子任务成功的终端状态。

## 2. 方法论：核心思想、关键技术细节、算法流程

- **核心思想**：在残差强化学习框架下，用离线学习的视觉远见预测器（foresight predictor）来重塑子任务奖励，引导残差策略产生更高“远见价值”的终端状态。
- **关键技术细节**：
  1. **远见价值函数**：定义为从当前子任务终端状态s出发，下游策略能实现的最终任务期望回报。对于子任务k，其学习目标变为局部远见价值函数：\( V_k^\circ(s) = E_{\pi_k}\left[ \sum_t \gamma^t \beta_k(s_{t+1}) V_{k+1:K}(s_{t+1}) \right] \)，即稀疏成功奖励乘以下游远见值。
  2. **离线远见预测器**：对于每个子任务边界，收集N个基础VLA策略成功完成的终端状态（图像+状态），然后对每个状态执行K_rep次下游策略回滚，记录下游成功的次数κ_i，形成标签y_i=κ_i/K_rep。再用二项负对数似然损失训练一个神经网络预测器p_ϕ: image → [0,1]，估计下游成功概率。
  3. **反向远见归纳训练**：按逆序（从最后一个子任务到第一个）训练残差策略。对于最后一个子任务（k=K），使用标准稀疏奖励；对于k < K，将预测器输出p_ϕ作为奖励乘数：\( r_t = \beta_k(s_{t+1}) \cdot p_ϕ(o_{t+1}) \)。这种逆序方式保证了下游策略固定，使远见估计平稳。
  4. **残差策略结构**：基策略为微调后的π0，残差策略仅接收腕部相机图像（通过DINOv2+LSTM），输出小幅度动作修正（α=0.1），且Actor输出层零初始化以保持初始行为。
- **算法流程**（文字描述）：
  - 步骤1：对每个子任务，微调π0获得基础策略。
  - 步骤2：对每个子任务边界（k→k+1），收集基础策略的成功终端状态，回滚下游策略，训练视觉远见预测器（Algorithm 2）。
  - 步骤3：逆序训练：先训练最后一个子任务（k=K）的残差策略（标准奖励）；然后对于k=K-1,...,1，加载已训练的下游策略，使用预测器输出作为奖励乘数，训练当前子任务的残差策略（Algorithm 1）。

## 6. 主要结论与发现

1. **标准残差RL提升单个子任务成功率但无法转化为全任务收益**：在oracle初始下，+Residual使插入成功率从45.7%提升到92.2%，但全任务成功率仅为54.5%，甚至不如end-to-end π0（54.9%）。这说明**改进子任务成功率而不控制终端状态分布，对组合性能无益**。
2. **远见残差RL显著提高链式任务成功率**：全任务成功率从54.5%（标准residual）提升至85.6%，提升31.1%。主要收益集中在Grasp→Insert边界（+31.8%）。
3. **远见残差RL改善了终端状态质量**：预测器评分显示，标准残差RL反而降低了终端状态的远见得分（Grasp终端从0.64降至0.32），而远见校正恢复并提升至0.73。这直接解释了表II的差异。
4. **定性分析**：远见校正的抓取更居中、角度更好，为插入提供更大裕度；远见校正的插入更牢固，能抵抗旋转力矩。而标准残差产生的终端状态在几何上成功但在接触动力学下脆弱。

## 7. 优点

- **方法创新**：精准识别了子任务耦合问题的本质——终端状态质量而非简单成功与否，并提出了理论上合理的解决方案（远见价值函数）。
- **实用性强**：离线预测器训练一次即可，在线RL时无需额外模拟，避免了高方差和高计算开销；与冻结VLA基础策略兼容，易于集成到现有流水线。
- **实验设计合理**：通过多个粒度评估（子任务、连续、全任务）清晰展示了“孤立改进无法迁移”的现象，并使用预测器评分和定性示例提供了因果解释。
- **对社区启示**：强调长时域操作需关注“哪些成功状态被产生”，为后续技能组合研究指明了方向。
