## 摘要
我们提出了FabriVLA，一种用于精准多任务操作的轻量级视觉-语言-动作模型。FabriVLA将InternVL3.5视觉-语言骨干网络与一个流匹配动作头相结合，该动作头具有跨动作令牌的门控自注意力机制和浅层VLM层融合，以丰富空间上下文。该模型通过单阶段联合优化进行训练，从预训练的VLM和随机初始化的动作头开始。在涵盖50个多样化操作任务的Meta-World MT50基准测试中，FabriVLA实现了90.0%的层级平均成功率，表明基于1B规模VLM构建的紧凑型VLA无需依赖数十亿参数的VLA骨干网络即可实现强劲性能。

## Abstract
We present FabriVLA, a lightweight Vision-Language-Action model for Precise Multi-Task Manipulation. FabriVLA combines an InternVL3.5 vision-language backbone with a flow-matching action head featuring gated self-attention across action tokens and shallow VLM layer fusion for enriched spatial context. The model is trained via single stage joint optimization from a pretrained VLM and randomly initialized action head. On the Meta-World MT50 benchmark spanning 50 diverse manipulation tasks, FabriVLA achieves a tier-average success rate of 90.0%, demonstrating that a compact VLA built on a 1B scale VLM can achieve strong performance without relying on multi billion parameter VLA backbones.

## 2. 方法论：核心思想与关键技术细节
### 核心思想
- **单阶段联合训练**：从预训练的VLM和随机初始化的动作头开始，整个模型端到端联合优化，所有参数解冻。
- **流匹配（Flow Matching）生成动作**：将动作生成视为从噪声到目标动作的连续概率流，通过预测速度场进行积分。
- **门控自注意力（Gated Self-Attention）**：在动作头的自注意力层中加入可学习的门控参数（初始化为0），训练时逐渐打开，允许动作令牌间建立时间依赖。
- **浅层VLM层融合（Shallow Layer Fusion）**：将VLM的深层语义特征（第14层）与浅层空间细节特征（第6层）拼接并线性投影，增强动作头对空间定位和接触操作的感知。

### 架构组件
1. **视觉语言骨干**：InternVL3.5-1B，保留14个transformer层，输入图像448×448。
2. **状态与动作编码器**：
   - 机器人状态（24维）通过两层MLP编码为1024维状态令牌，拼接到VLM上下文令牌前。
   - 噪声动作（50步×24维）通过共享MLP编码为动作令牌序列，并加位置编码。
3. **流匹配动作头**：
   - **8个FabriVLA Transformer Block**：每个块包含：
     - 门控自注意力（A′ = A + g·SelfAttn(LN(A))，g初始为0）
     - 交叉注意力到上下文
     - 带时间嵌入的前馈网络（FFN + TimeEmb）
   - **浅层融合**：将第6层和第14层VLM特征拼接后通过线性层投影回1024维。
   - **输出**：最终层归一化、序列池化、MLP解码为预测速度场 v ∈ R⁵⁰×²⁴。
4. **训练损失**：流匹配损失 L = E[∥vθ(xt, t|·) - (a-ε)∥²]，其中 xt = (1-t)ε + t a，t ~ Beta(2,2)，ε~U([-1,1])。
5. **推理**：从均匀噪声开始，用欧拉法积分50步得到动作，然后去归一化并执行。
