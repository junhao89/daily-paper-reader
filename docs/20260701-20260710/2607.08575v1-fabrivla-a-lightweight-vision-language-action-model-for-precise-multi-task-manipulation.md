## 摘要
我们提出了FabriVLA，一种用于精确多任务操作的轻量级视觉-语言-动作模型。FabriVLA将InternVL3.5视觉-语言骨干网络与流匹配动作头相结合，该动作头具有跨动作令牌的门控自注意力机制以及浅层VLM层融合，以丰富空间上下文。该模型通过单阶段联合优化，从预训练的VLM和随机初始化的动作头进行训练。在涵盖50个不同操作任务的Meta-World MT50基准测试中，FabriVLA达到了90.0%的层级平均成功率，表明基于1B规模VLM构建的紧凑型VLA无需依赖数十亿参数的VLA骨干网络即可实现强劲性能。

## Abstract
We present FabriVLA, a lightweight Vision-Language-Action model for Precise Multi-Task Manipulation. FabriVLA combines an InternVL3.5 vision-language backbone with a flow-matching action head featuring gated self-attention across action tokens and shallow VLM layer fusion for enriched spatial context. The model is trained via single stage joint optimization from a pretrained VLM and randomly initialized action head. On the Meta-World MT50 benchmark spanning 50 diverse manipulation tasks, FabriVLA achieves a tier-average success rate of 90.0%, demonstrating that a compact VLA built on a 1B scale VLM can achieve strong performance without relying on multi billion parameter VLA backbones.

## 2. 方法论

### 2.1 核心思想
- 将**预训练的视觉-语言模型（InternVL3.5-1B）**与**流匹配（Flow Matching）动作生成头**结合。
- 引入**门控自注意力（Gated Self-Attention）**：让动作令牌在预测视界内互相交互，但初始时门控为零（禁止自注意力），训练中逐渐打开，实现平滑学习。
- **浅层VLM层融合**：将VLM骨干的深层（语义）和浅层（空间细节）特征通过可学习线性投影融合，为动作头提供更丰富的空间上下文。

### 2.2 关键技术细节
1. **视觉-语言骨干**：采用InternVL3.5-1B，保留14层transformer。输入图像448×448，生成视觉令牌，与任务文本令牌拼接。
2. **状态与动作编码**：机器人状态（24维）通过两层MLP编码为1024维状态令牌，与VLM上下文拼接。动作令牌（50步×24维）通过共享MLP并加上位置编码得到50×1024维。
3. **动作头核心模块**：堆叠8个FabriVLA TransformerBlock，每个块包含：
   - **门控自注意力**：`A' = A + g * SelfAttn(LayerNorm(A))`，其中g初始化为0，可学习。
   - **交叉注意力**：`A'' = A' + CrossAttn(LayerNorm(A'), C, C)`，C为VLM上下文。
   - **时间条件FFN**：`A''' = A'' + FFN(LayerNorm(A'') + TimeEmb(t))`，时间嵌入使用正弦编码。
4. **浅层融合**：将VLM第6层（浅层）和第14层（深层）的特征沿特征维度拼接，再通过线性层投影回1024维，初始化权重为`[I | 0]`以保证与仅用深层时的功能一致。
5. **流匹配训练**：
