## 摘要
世界模型可以启用模型预测控制（MPC），但这需要动力学预测既要足够快以用于在线使用，又要具有足够的表达能力以表示不确定的未来。扩散模型为建模不确定动力学提供了一种自然机制，但其迭代推理过程使其难以用于低延迟的潜在规划。我们通过价值扩散世界模型（Valdi）弥合了这一差距，将用于MPC的端到端在线训练与潜在扩散动力学模型相结合。在CarRacing环境上的初步实验中，我们展示了Valdi在训练和推理时仅使用单个扩散步骤即可与确定性MLP基线相匹配。我们的实验揭示了在这种设置下预测多模态性与控制性能之间的权衡。代码可在https://github.com/Kit115/ValueDiffusionWorldModels获取。

## Abstract
World models can enable Model Predictive Control (MPC), but this requires dynamics prediction that is both fast enough for online use and expressive enough to represent uncertain futures. Diffusion models offer a natural mechanism for modeling uncertain dynamics, yet their iterative inference procedure makes them difficult to use for low-latency latent planning. We bridge this gap with Value Diffusion World Models (Valdi), combining end-to-end online training for MPC with a latent diffusion dynamics model. In preliminary experiments on the CarRacing environment, we show that Valdi, using a single diffusion step at both training and inference, matches a deterministic MLP baseline. Our experiments expose a trade-off between predictive multimodality and control performance in this setup. Code is available at https://github.com/Kit115/ValueDiffusionWorldModels.

### 2. 方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：在TD-MPC风格的TOLD框架中，将原本的确定性MLP动力学模型替换为潜在扩散动力学模型（双向编码器Transformer）。训练时使用单步去噪（one-step denoising），保持与推理时一致的低延迟。

  - **训练目标**：联合优化扩散损失 \(\mathcal{L}_{\text{diff}}\)（速度参数化，类似Salimans & Ho 2022）、奖励预测损失 \(\mathcal{L}_{\text{rew}}\)、时序差分损失 \(\mathcal{L}_{\text{val}}\)（用于价值学习）以及正则化项（SIGReg，防止潜在崩溃）。所有损失基于单步去噪后的潜在轨迹。
  - **算法流程**（伪代码见论文算法1）：
    1. 从回放缓冲区采样轨迹段 \((s_{t:t+H}, a_{t:t+H-1}, r_{t:t+H-1})\)。
    2. 用编码器 \(E_\theta\) 和目标编码器 \(\bar{E}_\theta\) 编码当前和未来状态，获得干净潜在 \(z\) 和 \(\bar{z}\)。
    3. 采样扩散时间步 \(\tau\) 和噪声，对 \(\bar{z}_{t+1:t+H}\) 进行正向加噪，得到 \(\bar{z}^\tau_{t+1:t+H}\)。
    4. 扩散模型预测速度 \(\hat{u}^\tau_{t+1:t+H}\)，并利用公式 \(\hat{z}_{t+1:t+H} = \sqrt{\alpha_\tau} \bar{z}^\tau_{t+1:t+H} - \sqrt{1-\alpha_\tau} \hat{u}^\tau_{t+1:t+H}\) 重建去噪后的潜在轨迹。
    5. 对重建潜在计算奖励损失和TD损失，加上扩散损失与正则化，更新在线网络，并通过指数移动平均（EMA）更新目标网络。

### 6. 主要结论与发现

1. **控制性能匹配**：单步扩散的Valdi在CarRacing环境中的控制性能与确定性MLP基线相当（差距在run-to-run方差内）。
2. **多模态预测能力**：增加推理扩散步数（2/4/8步）可以显著提高生成轨迹的视觉多样性（LPIPS升高），但**反而导致控制性能下降**（也在方差范围内），表明预测多模态性与控制质量之间存在权衡。
3. **价值函数特性**：在短规划视野下，Valdi的自洽性和与真实状态的偏移误差（grounding）不如MLP；但随着规划深度增加（接近规划器使用的终端步骤），Valdi的误差增长更慢，在终端步骤（h=4自洽性，h=5 grounding）上优于MLP基线。说明扩散模型的长视野价值预测更可靠。
4. **训练-推理不匹配**：单步扩散训练与多步扩散推理之间的不匹配是性能下降的可能原因；此外，CEM求解器难以利用增大的轨迹方差。

### 7. 优点

- **方法创新性**：首次将潜在扩散动力学模型端到端地集成到值引导的在线MPC框架（TD-MPC风格）中，且仅用单步扩散即可实用，实现了低延迟。
- **训练与推理一致性**：采用单步去噪同时用于训练和推理，避免了传统扩散模型迭代推理的高开销，是平衡速度与表达能力的有效设计。
- **揭示关键权衡**：明确指出了扩散世界模型中预测多模态性与控制性能之间的矛盾，为后续研究提供了方向（如何利用多模态而不损害规划）。
- **实验设计清晰**：对比基线严格控制变量，仅改变动力学模型，所有其他组件和超参数相同，确保比较公平。
- **价值函数诊断**：设计了两项诊断指标（自洽性和grounding），深入分析了不同动力学模型对价值学习的影响，具有方法论参考价值。

### 8. 不足与局限

- **环境单一**：仅在CarRacing上验证，该环境动态相对简单（有限随机性、部分可观测但可预测性较高）。未在更复杂、真正需要多模态建模的环境（如随机力、不规则交互、高维视觉）中测试，结论泛化性存疑。
