## 摘要
视觉-语言-动作（VLA）模型旨在将多模态输入映射到机器人动作。然而，现有方法大多因均匀处理所有视觉标记并使用人工选择因素进行推理而难以覆盖复杂动态场景，缺乏强调任务关键证据的机制，且忽略了潜在因素。为解决此问题，我们提出LEEVLA，一种在潜在环境演化中关注关键信息的VLA架构，它显式引导模型关注信息丰富区域，同时保持潜在世界表征的结构化演化。为识别显著且与指令相关的区域，我们引入漂移引导的动态优先级排序（DGDP），该方法结合动态位置优先排序（DPP）与语义漂移引导（SDG），指导VLA代理在训练中的关注点。在此基础上，我们引入结构化特征流生成（SFFG），通过原型到外围（P2P）预测建模这些优先级特征在潜在空间中应如何演化，并采用互邻域对比（MC）损失保持邻域间的拓扑一致性。DGDP与SFFG共同构成任务感知的“何处-如何”训练框架。在VLA基准上的大量实验表明，LEEVLA持续优于先前方法，证实显式任务证据引导与结构化潜在推理对于可扩展VLA均至关重要。我们的代码位于https://github.com/LyuQi127/LEEVLA。

## Abstract
Vision-language-action (VLA) models aim to map multimodal inputs to robot actions. However, most existing approaches struggle to cover complex dynamic scenarios due to treating all visual tokens uniformly and reasoning with human-selected factors, which lack mechanisms to emphasize task-critical evidence and ignore underlying factors. To address this issue, we propose LEEVLA, a VLA architecture for seeing what matters in Latent Environment Evolution that explicitly guides the model toward informative regions while preserving the structured evolution of latent world representations. To identify salient and instruction-relevant regions, we introduce drift-guided dynamic prioritization (DGDP), which combines dynamic position prioritization (DPP) with semantic drift guidance (SDG) to guide the VLA agent where to attend during training. On top of this, we introduce structured feature flow generation (SFFG), which models how these prioritized features should evolve in latent space via prototype-to-periphery (P2P) prediction, and a mutual-neighborhood contrastive (MC) loss to maintain topological consistency among neighborhoods. Together, DGDP and SFFG form a task-aware "where-how" training framework. Extensive experiments on VLA benchmarks show that LEEVLA consistently outperforms prior methods, confirming that explicit task-evidence guidance and structured latent reasoning are both crucial for scalable VLA. Our code is available at https://github.com/LyuQi127/LEEVLA.

- **问题**：现有视觉-语言-动作（VLA）模型在处理复杂动态场景时存在两个主要缺陷：
  - **均匀处理视觉 token**：对所有视觉特征一视同仁，无法突出与任务执行密切相关的区域（如被操作的物体、机械臂接触点）。
  - **依赖人工选择的已知因素**：如子目标图像、分割图、深度图等外部条件或特定线索进行推理，导致模型的搜索空间被限制，容易忽略任务相关的潜在未知因素（如物体在多模态下的变化、相似物体的区分）。
- **动机**：受一般视觉模型（DINOv2、SigLIP）的表征能力启发——这些模型的多模态特征已隐含语义、外观与几何信息。因此，论文提出**直接在潜在特征空间中进行推理**，将动作视为驱动环境状态在高维空间中演化的“引擎”，从而避免人工假设空间的局限。
- **整体含义**：通过显式引导模型“关注何处”（where to attend）并建模“如何演化”（how to evolve），实现更鲁棒、可扩展的 VLA 策略。

## 2. 方法论：核心思想与关键技术细节

### 2.1 整体框架
LEEVLA 采用“where–how”训练范式，包含两个核心模块，**仅在训练阶段使用，推理时不引入额外开销**：
- **漂移引导的动态优先级（Drift-Guided Dynamic Prioritization，DGDP）**：告诉模型“在哪里集中注意力”。
- **结构化特征流生成（Structured Feature Flow Generation，SFFG）**：告诉模型“这些特征应如何演化”。

### 2.2 DGDP：何处关注
- **动态位置优先（DPP）**：
  - 计算每个图像 patch 在相邻时间步（t 和 t+T）之间特征的**余弦相似度变化**，定义动态性得分 θᵢ = 1 – cos(v_{t,i}, v_{t+T,i})。
  - 得分高的区域（如被操作物体、机械臂）获得更高权重。
- **语义漂移引导（SDG）**：
  - 计算每个 patch 与语言指令的**相关性得分**（通过点积），并衡量其在时间步间的变化 Δᵢ = clip((r_{t+T,i} – r_{t,i}) / τ, –δ, δ)。
  - 通过指数函数和归一化得到语义漂移引导因子 ωᵢ，**放大向任务相关方向漂移的 patch**，抑制向背景漂移的 patch。
- **最终优先级权重**：βᵢ = σ(ωᵢ · θᵢ)，使用 sigmoid 函数。

### 2.3 SFFG：如何演化
- **原型到外围预测（P2P）**：
  - 对多视角未来特征（t+T 时刻）进行**聚类**，得到簇 C。
  - 在每个簇内，将成员按**到簇心的欧氏距离**排序（从近到远），形成有序序列 →c_ℓ。
  - 预测损失为加权余弦损失：L_{P2P} = 平均（(α + β_j) × [1 – cos(预测特征, 真实特征)]），其中 α=1 为全局因子，β_j 来自 DGDP 权重。
  - **作用**：保持同一语义单元（如一个物体）的特征在序列中连续预测，避免按固定光栅顺序破坏空间连续性。
- **互邻域对比损失（MC）**：
  - 构建高阶邻域图：对每个 anchor i，取一阶 Top-K 邻居 G⁽¹⁾ 和二阶邻居 G⁽²⁾（一阶邻居的 Top-M 邻居）。
  - 仅保留**互为邻居**的样本作为正样本对 G⁺，以抑制聚类噪声引入的虚假关联。
  - 使用 InfoNCE 损失拉近正样本对，保持局部拓扑一致性：L_{MC} = –1/|I| Σ_i 1/|G⁺| Σ_{j∈G⁺} log(exp(S_{ij}/τ_c) / Σ_{ℓ≠i} exp(S_{iℓ}/τ_c))。

### 2.4 训练目标
- 总损失：**L_total = λ₁ L_action + λ₂ L_P2P + λ₃ L_MC**。
- 动作损失采用 L1 回归（平行动作解码器），预测连续动作块。

## 6. 主要结论与发现

- **性能提升**：LEEVLA-mini（0.5B）和 LEEVLA-large（7B）在 LIBERO 和 CALVIN 上均达到 SOTA，分别以 97.5% 和 98.2% 平均成功率领先；CALVIN 平均任务长度达到 4.34，优于之前最佳（π0.5 的 4.02）。
- **组件互补**：DGDP（DPP+SDG）与 SFFG（P2P+MC）协同作用，单独使用均带来提升，联合使用产生最大增益。
- **结构化潜在推理有效**：通过原型-外围预测和互邻对比损失保持特征空间拓扑，明显优于平铺序列预测。
- **推理无额外成本**：DGDP 和 SFFG 仅在训练阶段激活，推理时无延迟增加。
- **显式任务证据引导至关重要**：模型若不关注动态且语义相关区域，性能显著下降（基线 94.8% → 完整 97.0%）。
