## 摘要
主流的语言-视觉-动作（VLA）模型在马尔可夫假设下主要根据当前观测预测动作，因此在处理长周期、时间依赖的任务时表现不佳。现有的记忆增强型VLA要么扩展观测窗口，要么从记忆库中检索历史信息作为辅助策略上下文。然而，它们将记忆置于VLA推理的原生潜嵌入空间之外，阻碍了历史经验与多模态推理及动作形成的流畅交织。为此，我们提出LaMem-VLA，一种原生潜记忆框架，将历史经验重构为潜记忆令牌，并直接与VLA推理交织。其核心是四个协调组件：（i）一个策展器，将历史经验组织为互补的短期和长期记忆库；（ii）一个搜索器，利用多模态认知查询两个记忆库以检索上下文相关证据；（iii）一个冷凝器，将检索到的证据重构为紧凑的短期和长期潜记忆令牌；以及（iv）一个编织器，将这些记忆令牌与当前观测和指令注入到一个连续的嵌入序列中。通过完全在相同的连续潜空间中表示、检索和消费历史经验，LaMem-VLA使记忆能够直接参与VLA推理并在有限上下文下指导动作生成。在SimplerEnv和LIBERO上的大量实验证明了我们LaMem-VLA的优越性。

## Abstract
Mainstream Vision-Language-Action (VLA) models predict actions primarily from the current observation under a Markovian assumption, thus struggling with long-horizon, temporally dependent tasks. Existing memory-augmented VLAs either expand the observation window or retrieve history from the memory bank as auxiliary policy-side context. However, they leave memory outside the native latent embedding space of VLA reasoning, preventing historical experience from being fluidly interleaved with multimodal reasoning and action formation. To this end, we introduce LaMem-VLA, a latent-memory-native framework that reconstructs historical experience into latent memory tokens and directly interweaves them with VLA reasoning. At its core, LaMem-VLA introduces four coordinated components: (i) a curator that organizes historical experience into two complementary short-term and long-term memory vaults; (ii) a seeker that queries both vaults using the multimodal cognition to retrieve context-relevant evidence; (iii) a condenser that reconstructs the retrieved evidence into compact short-term and long-term latent memory tokens; and (iv) a weaver that injects these memory tokens with the current observation and instruction into one continuous embedding sequence. By representing, retrieving, and consuming historical experience entirely in the same continuous latent space, LaMem-VLA enables memory to directly participate in VLA reasoning and guide action generation under a bounded context. Extensive experiments on SimplerEnv and LIBERO demonstrate the superiority of our LaMem-VLA.

- **核心问题**：如何将历史经验设计为**与VLA推理兼容的原生潜记忆**，使其能够被存储、检索并融入到与当前观测、语言指令相同的连续嵌入空间中，从而无缝参与动作推理。

### 论文提出的方法论

#### 核心思想
LaMem-VLA提出一种**原生潜记忆框架**，将历史经验重构为**短期和长期潜记忆令牌（latent memory tokens）**，并与当前观测和指令共同组成连续嵌入序列，使记忆直接参与VLA模型的内部推理过程。

#### 关键技术细节（四个协调模块）

1. **潜记忆策展器（Latent Memory Curator）**  
   - 将历史经验组织为两个互补的记忆库：  
     - **短期记忆库** \(M_{\text{short}}\)：存储当前回合的视觉感知证据（如物体位置、状态变化），以键-值对形式保存，键为平均池化后的压缩表示，值为经SE瓶颈压缩模块 \(C_s\) 处理后的视觉令牌。  
     - **长期记忆库** \(M_{\text{long}}\)：存储动作隐藏状态（action hidden states），记录任务进度和动作连续性信息，直接保存每步的 \(H_{\text{action}}\)。  
   - 各记忆库容量上限为 \(L=16\)，超出时通过计算相邻单元余弦相似度，合并最冗余的一对（取平均）以维持容量。

2. **潜记忆搜索器（Latent Memory Seeker）**  
   - 从VLA骨干（7B Prismatic VLM）输出的当前多模态认知表示 \(Q_{\text{con}}^t\)（融合视觉令牌 \(X_t\) 和指令令牌 \(I\)）出发，通过轻量Transformer查询构建器 \(B\) 生成上下文查询 \(Q_t\)。  
   - 使用平均池化后的全局查询向量 \(q_t\)，对短期记忆库按余弦相似度检索前 \(K=8\) 个单元，对长期记忆库同样检索前 \(K\) 个单元，得到检索集 \(Z_{\text{short}}\) 和 \(Z_{\text{long}}\)。

3. **潜记忆冷凝器（Latent Memory Condenser）**  
   - 引入可学习的短期视觉记忆槽 \(T_s \in \mathbb{R}^{L_s \times C}\) 和长期语义记忆槽 \(T_l \in \mathbb{R}^{L_l \times C}\)。  
   - 使用两个轻量Transformer记忆形成器 \(F_v\) 和 \(F_c\)，以查询令牌 \(Q_t\) 和检索证据为条件，更新记忆槽，输出固定长度的潜记忆令牌：  
     \[
     M_{\text{short}} = F_v([Q_t; Z_{\text{short}}; T_s])[-L_s:], \quad M_{\text{long}} = F_c([Q_t; Z_{\text{long}}; T_l])[-L_l:]
     \]  
   - 默认 \(L_s=8, L_l=4\)，确保注入序列的长度固定，独立于检索数量。

4. **潜记忆编织器（Latent Memory Weaver）**  
   - 将压缩后的短期/长期潜记忆令牌与当前视觉令牌、指令令牌和可学习动作查询令牌拼接，形成完整的记忆增强输入序列：  
     \[
     S_t = [M_{\text{short}} + \mathbf{1}_{L_s}\mathbf{b}_s^\top; \; M_{\text{long}} + \mathbf{1}_{L_l}\mathbf{b}_l^\top; \; X_t; \; I; \; Q_{\text{action}}]
     \]  
     其中 \(\mathbf{b}_s, \mathbf{b}_l\) 为可学习的源嵌入。  
   - 该序列经过VLM骨干，得到记忆增强的动作令牌 \(Z_{\text{action}}\)，然后送入扩散动作专家（diffusion-based action expert），通过DDIM采样子生成7自由度连续动作块（chunk size=16）。

### 论文的主要结论与发现

1. **性能超越**：LaMem-VLA在SimplerEnv-Bridge上以73.9%平均成功率超越所有对比方法（比CogACT高16.6%，比π0高4.7%）；在LIBERO上以97.6%平均成功率优于MemoryVLA（+1.1%）和CogACT（+4.4%），尤其在长周期套件Long-10和Long-90上分别达到95.8%和97.0%。  
2. **双记忆互补**：短期记忆保留视觉细节，长期记忆追踪任务进度，两者结合比单独使用效果更优。  
3. **原生潜整合优于外部策略侧条件**：将记忆令牌直接注入VLA推理序列比作为策略侧额外条件效果更好。  
4. **固定长度令牌优势**：通过冷凝器将检索证据压缩为固定长度令牌，避免了输入上下文无限增长，同时保持有效历史信息。

### 优点

- **创新性**：首次将历史经验完全表示为**与VLA推理同空间的潜记忆令牌**，实现记忆与感知、语言、动作的深度融合，而非附加的辅助条件。  
- **双尺度设计**：短期（视觉主导）和长期（语义/动作进度）记忆各自独立存储和压缩，适应不同时间尺度的需求。  
- **高效性**：冷凝器输出固定长度令牌，使推理上下文可控，不随历史增长而线性膨胀。  
- **实验结果优异**：在两个主流仿真基准上均达到最高成功率，且消融实验充分验证了各组件的必要性。  
- **代码框架公开**（项目页即将发布），便于复现和扩展。

### 不足与局限

- **实验环境限制**：所有评估仅在**仿真平台**（SimplerEnv和LIBERO）上进行，缺乏真实机器人平台验证，泛化到真实世界的效果未知。  
- **偏差风险**：仿真环境中的视觉和物理动态相对可控，可能低估了真实场景中的噪声、视觉变化和意外情况对记忆检索的挑战。  
- **计算开销**：虽然记忆令牌长度固定，但每次推理仍需经过搜索器、冷凝器等多个轻量Transformer模块，相较于无记忆基线增加了推理延迟，文中未报告具体速度指标。  
- **检索依赖**：搜索器基于当前观测和指令构建查询，若当前观测存在歧义或视觉干扰，可能导致检索到不相关历史，影响后续推理（文中未对此进行鲁棒性分析）。  
- **超参数敏感**：检索数量 \(K\)、记忆令牌长度 \(L_s, L_l\) 等需手动调节，最优配置可能因任务类型而异，缺乏自适应机制。  
- **长期记忆更新策略**：当前采用简单的相邻余弦相似度合并冗余，可能丢弃关键但非连续的变化信号。
